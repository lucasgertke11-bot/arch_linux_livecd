# Arch Linux Live CD Base

Base mínima do Arch Linux para DistroFrogger.

## Download

Baixe o release `arch_base.tar.bz2` na aba [Releases](https://github.com/lucasgertke11-bot/arch_linux_livecd/releases).

## Uso

```bash
tar -xjf arch_base.tar.bz2
cd "arch base"
# Pronto para usar como base do seu sistema
```
